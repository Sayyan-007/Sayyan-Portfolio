● Border

*border-color = color as a …primary and etc...….
*border-number = the number is how much that given for the thickness of border
*additive border =(border-position) to give the border on that position only  
*subractive border =(border-position-0) to remove the border on that position only
nb:-(positons=top,bottom,start and end)
● Border-Radious
*rounded-top/bottom/start/end

● Margin and Padding

*m-size=will make margin from all position
*mt-size=will make top margin only
*mb-size=will make bottom margin only 
*ms-size=will make start/left side margin only
*me-size=will make top end/right margin only
*mx-make start and end margin only
*my-make top and bottom margin only

*p-size=will make padding from all position
*pt-size=will make top padding only
*pb-size=will make bottom padding only 
*ps-size=will make start/left side padding only
*pe-size=will make top end/right padding only
*px-make start and end padding only
*py-make top and bottom padding only

nb:-size should be in 0-5
